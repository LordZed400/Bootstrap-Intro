Introduction
Bootstrap is a front-end development tool that helps us to stylize our Web Document or Web Application. It makes front end development faster and easier by helping in developing responsive, mobile-first projects on the Web.
It is a popular framework for HTML, CSS and JS.

It is a Mobile first framework, meaning it is used with the mobile display in mind first, and then only moving towards Desktop environment. It helps in creating Responsive Design of an Application. Responsive means that the webpage will automatically adjust itself to appear proper whether it is in a desktop browser or a mobile browser. This was a step up to the previous concept of "Adaptive Design" where multiple web-pages were made which corresponded to individual environments like mobile, tablet or desktop. Then one page out of the numerous pages were according to the environment or resolution.

Responsive

Responsive and Adaptive are two approaches to creating a Webpage or Web Application. These are techniques of handling different environments or different browser resolutions to make the user experience better.

Responsive

Responsive Designs allow us to navigate through the Webpage across multiple screens and Browsers with ease. They don't use dedicated mobiles websites, rather they utilise flexible grids and layouts with proper use of CSS. This optimizes the website and allows streamlined viewing of content over all resolutions.

Non-Responsive

Non-Responsive Designs rely on dedicated webpages, meaning for every possible device out there (ranging from Desktop to Small Screen Mobile phones) there is a separate website, ultimately hindering its realtionship with Google for not using the same URL across multiple devices.

But there is also a third approach called a Hybrid approach, that combines Adaptiveness and Responsiveness to create the best user experience.
